🧠 Project Plan: Fullstack Todo App (MERN + Tailwind)
⚙️ Tech Stack (Phase 1)
Frontend: React (JavaScript), Tailwind CSS
Backend: Express.js
Database: MongoDB (Mongoose)
API: REST
Later phases: TypeScript, auth, deployment, tests
🪜 Development Phases
✅ Phase 1 — Backend API (Express + MongoDB)

We build:

MongoDB connection
Todo model
REST API
Endpoints:
Method	Route	Description
GET	/api/todos	Get all todos
POST	/api/todos	Create todo
PUT	/api/todos/:id	Update todo
DELETE	/api/todos/:id	Delete todo

Todo list UI
